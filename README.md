The Technologies and Languages used in this project

- Javascript
- HTML
- CSS & Bootstrap
- ReactJs for Front-End
- NodeJs for Back-End
- MySQL for Database

Steps to run the project

1- clone the git.
2- create database named "milele_db"
3- import SQL file located at "back-end/milele_db.sql"
4- run "cd back-end"
5- run "npm i"
6- run "npm start"
7- run "cd.. "
8- run "cd front-end"
9- run "npm i"
10- run "npm start"
